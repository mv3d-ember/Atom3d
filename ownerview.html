Setup — Visitor Logging
1
Go to supabase.com → New project (free, no credit card). Once created, go to Settings → API and copy your Project URL and anon public key.
2
In Supabase, open SQL Editor and run this once to create the visitors table:
create table visitors ( id bigserial primary key, ip text, city text, region text, country text, isp text, browser text, os text, device text, screen text, viewport text, lang text, tz text, referrer text, page text, ts timestamptz default now() ); alter table visitors enable row level security; create policy "anon_insert" on visitors for insert to anon with check (true); create policy "anon_select" on visitors for select to anon using (true);
3
Paste your Project URL and anon key below and hit Save. Tracking will activate automatically on atom3d.org.
Stored only in this browser's localStorage. Tracking activates on atom3d.org immediately — no deploy needed. Your own visits (owner IP) are never logged.
